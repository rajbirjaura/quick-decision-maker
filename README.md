# Quick Decision Maker

A full-stack poll app where users can create polls, vote, and see results.

---

# Features

- Create polls (2–4 options)
- Vote on polls
- One vote per user
- Poll expiration timer
- Winner display
- Delete polls
- Modern UI (Tailwind CSS)

---

# Tech Stack

- React
- Node.js
- Express
- MongoDB
- Tailwind CSS

---

# Setup Instructions

# 1. Install backend

cd server  
npm install  
npm start  

---

# 2. Install frontend

cd client  
npm install  
npm run dev
